### 小红书 B版（专业向）

**标题**：Claude Code /insights：7 步流水线分析你的使用模式

**正文**：

**功能概述**
`/insights` 命令生成交互式 HTML 报告，分析你所有 Claude Code 会话的使用模式。报告保存在 `~/.claude/usage-data/report.html`

**分析流水线（7个阶段）**
1. 会话过滤：排除少于2条消息或不足1分钟的会话
2. 长对话摘要：超过 30K 字符的对话分段总结
3. 方面提取：用 Haiku 模型分析目标、满意度、摩擦点
4. 聚合分析：项目领域、交互风格、工作成效、改进建议
5. 执行摘要：什么好用、什么阻碍、快速改进、未来工作流
6. 报告渲染：统计仪表板 + 可视化图表 + 叙述分析

**报告包含内容**
• 统计：会话数、消息数、时长、Token 用量、Git 活动
• 图表：每日活动、工具使用分布、语言分布、满意度
• 建议：可复制粘贴的技能/代理/钩子代码

**隐私说明**
所有分析在本地完成，会话数据不离开你的机器。
