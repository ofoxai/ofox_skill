<summary>小红书B ｜ DeepSeek-V3.2技术解析 ｜ 算法效率超越算力堆叠</summary>

**标题**：DeepSeek-V3.2 技术解析：如何用更少算力超越 GPT-5

---

DeepSeek 发布 V3.2，高配版 Speciale 在推理基准上超越 GPT-5，与 Gemini-3.0-Pro 持平。

**三个核心技术突破**：

**1. DeepSeek Sparse Attention (DSA)**
将注意力计算复杂度从 O(n²) 降到 O(n)，长上下文场景获得显著端到端加速。这是支撑 128K 上下文的关键。

**2. 大规模强化学习**
RL 阶段消耗的计算资源超过预训练本身。这与 OpenAI 的 o1 思路类似，但 DeepSeek 在开源框架下实现。

**3. 专家蒸馏 + 合成数据**
训练专门的编程、数学、Agent 模型，用它们生成高质量合成训练数据。这解决了高质量训练数据稀缺的问题。

**开发者视角**：
- 模型权重在 Hugging Face 开放下载（MIT 协议）
- Speciale 高配版仅通过 API 提供
- 128K 上下文，适合长文档和复杂代码分析
- 世界知识广度仍逊于闭源模型

**权衡**：推理能力强，通用知识稍弱。适合代码、数学、工具调用场景。
