### 小红书 A版（洞察向）

**标题**：OpenAI 花了 100 亿美元，就为不再只靠 Nvidia

**正文**：

你知道 AI 行业最大的"卡脖子"问题是什么吗？

不是算法，不是数据，是算力——准确说，是 Nvidia 的 GPU。

**发生了什么**

OpenAI 和 Cerebras 签了一份 100 亿美元的算力协议。Cerebras 将在 2028 年前向 OpenAI 提供 750 兆瓦的计算能力，成为全球最大的高速 AI 推理部署。

**这笔交易的重点不是钱**

而是一个信号：AI 巨头们开始认真"去 Nvidia 化"了。

1. **Cerebras 的芯片有多不一样**
它做的是"晶圆级芯片"——一整块晶圆就是一颗芯片，推理速度比 GPU 方案快 15 倍。

2. **为什么是推理而非训练**
训练模型是一次性的，推理是每天每秒都在发生的。ChatGPT 每天处理几亿次请求，推理成本才是大头。

3. **供应链安全的考量**
把鸡蛋都放在 Nvidia 的篮子里，风险太大。这笔交易是 OpenAI 分散供应链的关键一步。

**So What**

对我们普通用户来说：AI 推理更快 = ChatGPT 响应更快 = 体验更好。对行业来说：算力供应多元化意味着 AI 成本有望进一步下降。

### 小红书 B版（专业向）

**标题**：OpenAI 签 Cerebras $100亿协议：推理侧"去 Nvidia 化"正式开始

**正文**：

OpenAI 和 Cerebras 的这笔交易值得仔细看。不只是大数字，背后是 AI 基础设施格局的关键变化。

**交易结构**

| 项目 | 细节 |
|------|------|
| 金额 | $100亿+ |
| 期限 | 至 2028 年，分阶段部署 |
| 算力 | 750 兆瓦计算能力 |
| 模式 | Cerebras 建/租数据中心 → OpenAI 使用云服务 |
| 定位 | 推理（Inference），非训练 |

**为什么选 Cerebras**

Cerebras 的核心卖点是晶圆级引擎（Wafer-Scale Engine）：

- **单芯片面积**：整块 300mm 晶圆 = 一颗芯片
- **推理速度**：比 GPU 方案快 15x
- **延迟优势**：低延迟推理 → 更自然的实时交互

对 OpenAI 来说，这解决的是推理侧的瓶颈。训练可以排队等，但 ChatGPT 每天数亿次请求的推理延迟和成本，直接影响用户体验和利润率。

**战略分析**

1. **算力供应链多元化**
OpenAI 此前已和 AMD、Broadcom 建立芯片合作。加上 Cerebras，"去 Nvidia 单一依赖"的意图非常明确。

2. **推理 vs 训练的分离趋势**
训练用 Nvidia GPU（CUDA 生态无可替代），推理用专用芯片（Cerebras/TPU 等）。未来 AI 公司可能普遍采用混合架构。

3. **Cerebras 的商业化转折点**
此前 87% 营收来自阿联酋 G42。拿下 OpenAI 意味着客户集中度风险大幅降低，IPO 前景更清晰。

**开发者影响**

- API 调用延迟可能进一步降低
- 推理成本下降 → API 定价可能调整
- 实时场景（语音对话、Agent 操作）的体验会明显改善

还记得 Meta 几年前砍掉新闻 Feed、不再给媒体分钱吗？

现在它又回来了。但不是因为良心发现，是因为 AI。

**发生了什么**

Meta 和 CNN、Fox News、USA Today、Le Monde 等主流媒体签了多年期授权协议。内容会被用在 Meta AI 聊天机器人里——当你在 WhatsApp、Instagram、Facebook 上问新闻时，AI 会引用这些媒体的内容来回答，并附上原文链接。

以前的搜索引擎是"链接分发器"，AI 搜索是"答案生成器"。内容不再是导流工具，而是 AI 回答的原材料。谁控制了优质内容，谁的 AI 回答就更准确。

2. **"免费白嫖"的时代正在结束**
OpenAI 被出版商起诉、Google 在欧洲被迫付费——现在 Meta 主动签约。整个 AI 行业正在从"先抓取再说"转向"先授权再用"。

不再靠广告点击活着，而是把内容授权给 AI 公司。美国新闻媒体联盟称这是"在漫长斗争中为出版商赢得的胜利"。

**So What**

对我们来说：AI 回答新闻问题时会更准确、有出处。对创作者来说：内容版权正在被重新定价——你的内容可能比你想的更值钱。

**交易细节**

| 项目 | 内容 |
|------|------|
| 合作方 | CNN、Fox News、USA Today、Le Monde、People Inc、The Daily Caller、Washington Examiner |
| 覆盖范围 | 全球、突发、娱乐、生活、体育新闻 |
| 期限 | 多年期协议 |
| 应用场景 | Meta AI 聊天机器人（Facebook/Instagram/WhatsApp/Messenger） |
| 呈现方式 | 引用内容 + 链接回原文 |

**技术视角：RAG 的内容供应链**

这笔交易本质上是在建 Meta AI 的 RAG（检索增强生成）内容供应链：

1. **实时性问题**
LLM 的知识截止于训练时间。新闻场景天然需要实时信息。授权协议解决了"用什么数据做 RAG"的合规问题。

2. **可信度标注**
AI 回答附带来源链接，降低幻觉问题的影响。用户可以点击验证。

3. **内容质量控制**
不是爬全网，而是选择性接入权威来源。这是 RAG 质量的关键决策。

**商业格局对比**

| 公司 | 内容授权策略 |
|------|-------------|
| OpenAI | 部分签约 + 被 NYT 等起诉 |
| Google | 欧洲被迫付费 + 搜索结果整合 |
| Meta | 主动签约多家主流媒体 |
| Perplexity | 被多家媒体起诉中 |

**开发者启示**

- **内容合规成本**：AI 产品涉及新闻/实时信息，内容授权正在成为必选项
- **RAG 架构设计**：需要考虑内容来源的合规性和可追溯性
- **Attribution 机制**：AI 回答标注来源正在成为行业标准
